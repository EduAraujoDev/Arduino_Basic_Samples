# Arduino_Basic_Samples / GSM_GPRS
Projeto com exemplo de uma protopitapação para GSM_GPRS, usando Arduino UNO. Para essa projetos, foi usado os seguintes componentes:

 - Arduino UNO
 - Jumper Macho-Femea
 - Módulo GSM GPRS SIM800L
