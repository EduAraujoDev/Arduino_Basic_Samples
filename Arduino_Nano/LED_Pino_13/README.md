# Arduino_Basic_Samples / Arduino_Nano  / LED_Pino_13
Projeto com exemplo simples usando led com arduino nano. Para essa projetos, foi usado os seguintes componentes:

 - Arduino NANO
 - Jumper Macho-Macho
 - Led Vermelho
 - Resistor 220R
