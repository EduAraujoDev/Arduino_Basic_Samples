# Arduino_Basic_Samples / Arduino_Nano  / LED_Pino_13_Blink
Projeto com exemplo simples usando led (blink) com arduino nano. Para essa projetos, foi usado os seguintes componentes:

 - Arduino NANO
 - Jumper Macho-Macho
 - Led Vermelho
 - Resistor 220R
