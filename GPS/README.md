# Arduino_Basic_Samples / GPS
Projeto com exemplo de uma protopitapação para GPS, usando Arduino UNO. Para essa projetos, foi usado os seguintes componentes:

 - Arduino UNO
 - Jumper Macho-Femea
 - Módulo GPS GY-NEO6MV2
