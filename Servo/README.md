# Arduino_Basic_Samples / Servo
Projeto com exemplo de uso com Servo, usando Arduino UNO. Para essa projetos, foi usado os seguintes componentes:

 - Arduino UNO
 - Jumper Macho-Femea
 - Servo TowerPro MG996
